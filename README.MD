This application uses NodeJS framework for backend and JavaScript EJS templating engine for client side rending in frontend. 

For storing Employers and Employees data, the application uses Mongodb. 

Oshi time also supports user authentication (LOGIN/LOGOUT) both on Employers side and Employees side.
